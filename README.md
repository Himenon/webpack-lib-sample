## Usage

### Setup

```sh
yarn
```

### Development

```sh
yarn start
```

### Production Build

```sh
yarn run build:prod

# check use http server
serve -s build
```

## Update

```sh
yarn outdated
yarn upgrade
```

## Publish

```
yarn publish
```
